# Chan-ce
2021 Mobile Programming project in ITM  
**"Chan:ce"** 는 동네 반찬 구독 서비스 어플리케이션입니다.  
본인이 원하는 음식점을 검색하고 반찬을 고른 후 반찬 수령일을 선택하여 구독을 확정할 수 있습니다.
   
# Tech Stack
- Kotlin
- Android Studio
- Firebase
